# Newsletter Boilerplate

A boilerplate for quick newsletters templates.

## How to use

1. Open `index.html`
1. Insert images in `\img` folder
1. Start hacking away

## Support

1. Gmail
1. Hotmail
1. Yahoo Mail
1. Outlook
